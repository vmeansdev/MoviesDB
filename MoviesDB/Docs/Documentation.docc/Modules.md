# Modules

- `MoviesDB`
  - App composition, scene wiring, coordinators, DI, and SwiftUI Watchlist.
- `Dependencies/MovieDBUI`
  - Reusable UIKit and SwiftUI components: list/grid UI, cells, buttons, view controllers, snapshot tests.
- `Dependencies/MovieDBData`
  - Models, services, caching, watchlist store.
- `Dependencies/AppHttpKit`
  - HTTP client abstraction and request/response models.
- `Dependencies/PlistReader`
  - Reads values from `Info.plist`.
