# Overview

- `MoviesDB` is the app target.
- `MovieDBUI` is the shared UI package.
- `MovieDBData` is the data/networking package.
- `AppHttpKit` is the HTTP client package.
- `PlistReader` is used for configuration in `Info.plist`.

Minimum deployment target is **iOS 18** and the app is built with **Swift 6**.

## Key Files
- `MoviesDB.xcodeproj`
- `MoviesDB/Resources/Info.plist`
